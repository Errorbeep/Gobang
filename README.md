# Gobang
五子棋游戏 CS架构
a Gobang game

### 技术栈
+ Java/JavaFX
+ CSS/FXML
+ Java Socket

### 主要功能
+ 判定输赢
+ 聊天
+ 悔棋
+ 重开
+ 局域网

### 运行环境
Java 8或者其它带有JavaFX的Java版本，如librica-java full
